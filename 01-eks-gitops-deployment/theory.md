# Theory Notes

## GitOps

GitOps is a deployment methodology where Git acts as the single source of truth, and tools like ArgoCD synchronize Kubernetes clusters with Git repositories.

## Helm

Helm is the package manager for Kubernetes. It packages Kubernetes resources into reusable charts.

## ArgoCD

ArgoCD continuously monitors Git repositories and deploys changes automatically to Kubernetes clusters.

## IRSA

IAM Roles for Service Accounts (IRSA) allow Kubernetes Pods to securely access AWS services without storing AWS access keys.

## AWS Secrets Manager

AWS Secrets Manager securely stores sensitive information such as passwords and API keys.

## ALB Ingress

Application Load Balancer (ALB) exposes Kubernetes applications to external users.

## Grafana

Grafana visualizes metrics collected from applications and infrastructure.
